MYKTEAM Private DNS Server

This server was created for my personal and family and friends to use. If you like it, you can also use it.

My servers support forwarding ECS IP addresses to upstream servers for access handling in order to provide the IP addresses of services as close to your place as possible. The defined ECS address will be a secure anonymous address belonging to your isp, sometimes it may cause the incorrect regional determination resulting in the receipt of a server ip that is not close to your region, this problem comes from your isp so there is nothing else i can do.

The server supports ad-blocking filters that are updated hourly, block trackers, pages that spread malware.

I hate phishing, so the domain data from the verified filter from the Vietnam Anti-Fraud Project ( https://chongluadao.vn/) is integrated and constantly updated.

I'm not sure if it already supports foreign phishing domains, if there is support congratulations you will also get this protection from the project.

Note: I don't like gambling games, so gambling-related domains if they are in the filter will be blocked. Sorry, can't do anything else.

In addition to the automatic lists, I also use manual allow/block lists. If you need to add a domain name to the list, please send me an email and provide me with the domain name to block/allow and the reason for adding it, I will verify and add it to the list if it is valid.

Automated firewalls can sometimes block you by mistake. If you are blanketed, please provide me with your IP address via email. I'm going to take it down on the firewall.

Privacy issues: I don't care what you access or where you comes from, nor do i force you to use this server. Logs are only used to check which domains need to be manipulated and manually add domains to the list.

Note: I work in an automation/application development and programming environment. Therefore, some domains will still be allowed manually for target debugging and error status monitoring, they may be reported as unblocked on ad and tracking blocking testing pages.

The list of ignored domains is updated by me here: Igrone List

Email contact : admin@mykteam.com
